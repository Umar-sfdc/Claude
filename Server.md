# MCP Remote Server

- Create a folder with your server name
- Install uv
- Open the folder in the vscode
- `uv init`
- `uv add fastmcp`
- Create a simple server
- Run the server `uv run main.py` or `fastmcp run main.py --transport http --host 0.0.0.0 --port 8000`
- Test the server using mcp inspector
- Test the server `uv run fastmcp dev main.py`
